
# Historia de Usuario
## ID
[FE-09]
## Título
Comunidad — foro de familias con posts y respuestas
## Descripción
**Como** padre de un niño con autismo  
**Se desea** poder publicar dudas, experiencias y leer las de otras familias  
**Para** sentirse acompañado y obtener consejos de quienes viven situaciones similares.
## Criterios de Aceptación
### Escenario 1: Listado de publicaciones
```gherkin
Dado que el usuario navega a "/comunidad"
Cuando la página carga
Entonces debe ver un listado de posts con: avatar del autor, nombre, fecha, título, preview del contenido, número de respuestas y likes
Y los posts deben estar ordenados por más recientes por defecto
```
### Escenario 2: Crear una publicación (usuario autenticado)
```gherkin
Dado que el usuario ha iniciado sesión y está en "/comunidad"
Cuando hace clic en "Nueva publicación"
Entonces debe abrirse un formulario o modal con campos de título, cuerpo del mensaje y etiquetas opcionales
Y al enviar, el nuevo post debe aparecer al inicio del listado sin recargar la página
```
### Escenario 3: Acceso restringido para no autenticados
```gherkin
Dado que el usuario NO ha iniciado sesión
Cuando intenta crear una publicación o escribir una respuesta
Entonces debe redirigirse a "/login" con un mensaje: "Debes iniciar sesión para participar en la comunidad"
Y debe conservarse la URL original para redirigirle de vuelta tras el login
```
### Escenario 4: Sistema de likes
```gherkin
Dado que el usuario autenticado visualiza un post
Cuando hace clic en el ícono de like (corazón)
Entonces el contador debe incrementarse inmediatamente (optimistic update)
Y si el usuario ya había dado like, debe deshacerlo al volver a hacer clic
```
## Notas
- Los posts deben soportar formato básico (negrita, cursiva, listas) con un editor simple tipo WYSIWYG o Markdown.
- Implementar paginación infinita (infinite scroll) o paginación tradicional.
- Considerar moderación: los posts deben poder ser reportados por los usuarios.
- Las notificaciones de respuestas se implementan en una historia posterior.
