# Historia de Usuario
## ID
[FE-01]
## Título
Header de navegación global responsivo
## Descripción
**Como** padre o visitante del sitio  
**Se desea** tener un menú de navegación claro y accesible en todas las páginas  
**Para** desplazarse fácilmente entre las secciones del portal sin perderse.
## Criterios de Aceptación
### Escenario 1: Renderizado del header en desktop
```gherkin
Dado que el usuario accede desde un dispositivo con pantalla mayor a 860px
Cuando carga cualquier página del sitio
Entonces debe visualizarse el logo, el nombre "patitas", el nav con 5 enlaces y los botones de búsqueda, carrito e iniciar sesión
Y el enlace de la sección activa debe tener fondo lila (#c58cf2) con texto blanco
```
### Escenario 2: Menú hamburguesa en móvil
```gherkin
Dado que el usuario accede desde un dispositivo con pantalla menor a 860px
Cuando hace clic en el botón hamburguesa
Entonces el menú desplegable debe aparecer con animación suave (max-height transition)
Y al hacer clic en cualquier enlace del menú, éste debe cerrarse automáticamente
```
### Escenario 3: Hover y estados interactivos del nav
```gherkin
Dado que el usuario pasa el cursor sobre un enlace de navegación no activo
Cuando hace hover sobre él
Entonces debe aplicarse un fondo lila suave (rgba(197,140,242,0.14)) sin cambiar el color del texto
Y el botón "Iniciar sesión" debe oscurecerse y elevarse 2px con sombra al hacer hover
```
### Escenario 4: Persistencia del header al hacer scroll
```gherkin
Dado que el usuario está navegando en la página
Cuando hace scroll hacia abajo más de 80px
Entonces el header debe quedar fijo (sticky) en la parte superior con sombra sutil
Y debe reducirse ligeramente el padding para no ocupar demasiado espacio vertical
```
## Notas
- El fondo del header usa `fondo_morado.png` con `background-size: cover`.
- El logo debe tener animación de scale + rotate al hacer hover.
- El anillo punteado animado del logo es decorativo y debe pausarse en `prefers-reduced-motion`.
- Los íconos de búsqueda y carrito usan Material Symbols Outlined.
- El wave inferior de separación debe usar el color de fondo de la página (`#f0f8ff`).
