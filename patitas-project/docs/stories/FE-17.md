## Título
Hero Banner — Botones CTA funcionales
## Descripción
**Como** visitante que llega a la página principal  
**Se desea** que los botones del hero banner ("Comenzar ahora" y "Conocer recursos") lleven a destinos reales y útiles  
**Para** comenzar a explorar la plataforma sin callejones sin salida.
## Criterios de Aceptación
### Escenario 1: Botón "Comenzar ahora" — usuario no autenticado
```gherkin
Dado que el visitante no tiene sesión iniciada
Cuando hace clic en "Comenzar ahora"
Entonces debe navegar a "/registro"
Y la página de registro debe cargarse correctamente con el formulario completo
```
### Escenario 2: Botón "Comenzar ahora" — usuario autenticado
```gherkin
Dado que el usuario ya tiene sesión iniciada
Cuando hace clic en "Comenzar ahora"
Entonces debe navegar a "/recursos"
Y la página de recursos debe cargarse con el catálogo de artículos, guías y vídeos
```
### Escenario 3: Botón "Conocer recursos"
```gherkin
Dado que cualquier visitante (autenticado o no) hace clic en "Conocer recursos"
Cuando el botón se activa
Entonces debe navegar siempre a "/recursos"
```
## Notas
- Reemplazar `<a href="#">` por `<RouterLink>` con lógica condicional basada en `authStore.isAuthenticated`.
- El componente `HeroBanner.vue` debe importar `useAuthStore` y `useRouter` para implementar la lógica.
- "Comenzar ahora" → `/registro` si no autenticado, `/recursos` si autenticado.
- "Conocer recursos" → siempre `/recursos`.
