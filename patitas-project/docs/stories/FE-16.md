# Historia de Usuario
## ID
[FE-16]
## Título
Modal de rastreo rápido de pedido desde la Tienda
## Descripción
**Como** cliente que está navegando la tienda  
**Se desea** poder consultar el estado de un pedido sin salir de la página ni bajar al footer  
**Para** acceder rápidamente a la información de envío y a las opciones de cambio o devolución.
## Criterios de Aceptación
### Escenario 1: Apertura del modal
```gherkin
Dado que el usuario está en "/tienda"
Cuando hace clic en el botón "Estado del pedido" del banner CTA
Entonces debe abrirse un modal centrado con un campo de código de pedido
Y el fondo de la página debe tener un overlay con blur
Y el modal debe cerrarse al hacer clic fuera de él o en el botón de cierre
```
### Escenario 2: Pedido encontrado
```gherkin
Dado que el usuario introdujo un código válido (ej: PT-48291) en el modal
Cuando hace clic en "Buscar" o presiona Enter
Entonces debe mostrarse dentro del modal: estado del pedido, fecha de creación, entrega estimada y total
Y deben aparecer dos botones: "Ver detalle completo" → "/pedido/:id" y "Solicitar cambio / devolución" → "/devoluciones?order=PT-XXXXX"
```
### Escenario 3: Código no encontrado
```gherkin
Dado que el usuario introdujo un código inexistente
Cuando hace clic en "Buscar"
Entonces debe mostrarse un mensaje de error rojo con ícono
Y debe aparecer un botón "Intentar de nuevo" que limpia el input
```
### Escenario 4: CTA dual en la tienda
```gherkin
Dado que el usuario está en cualquier parte de "/tienda"
Cuando observa el banner CTA inferior de la página
Entonces debe ver dos botones visibles sin hacer scroll: "Estado del pedido" (abre modal) y "Cambios y devoluciones" (navega a "/devoluciones")
```
## Notas
- El modal no requiere autenticación: cualquier usuario puede buscar por código de pedido.
- El campo de código aplica `toUpperCase()` automáticamente al escribir.
- Al navegar desde el modal a "/devoluciones", pasar `?order=PT-XXXXX` para pre-rellenar el pedido en el formulario.
- El estado del pedido en el modal muestra color semántico: amarillo (preparando), azul (en camino), verde (entregado).
